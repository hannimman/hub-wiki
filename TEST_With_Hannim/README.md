# 🧸 파츠 상점 — 캐릭터 꾸미기 토이 프로젝트

이미지 파일 0개! 캐릭터·아이템 144종을 전부 **코드(SVG)** 로 그린 웹 토이 프로젝트입니다.

## 실행 방법

```powershell
# 방법 1: 그냥 index.html을 더블클릭 (빌드 불필요)
# 방법 2: 정적 서버 (권장)
python -m http.server 8741
# → http://localhost:8741 접속
```

## 기능

- **얼굴 커스터마이징** (무료): 눈 8종 / 코 8종 / 입 8종
- **상점**: 12개 파츠 슬롯, **149개 아이템** (기본 120 + 코스프레 29)
  - 헤어, 모자, 얼굴 악세서리, 수염, 상의, 하의, 신발,
    왼손/오른손 아이템, 좌/우 바닥 장식, 배경
  - 카드 클릭 = **입어보기**(무료 미리보기), 구매 버튼 = 결제
- **코스프레 세트** 🎭: 슬롯별 파츠를 모아 장착하면 완성
  - 파란 로봇냥(도라에몽풍), 노란 전기쥐(피카츄풍), 동글귀 생쥐(미키풍),
    세일러 오리(도널드풍), 노란 스폰지(스폰지밥풍) — 각 세트는 모자/얼굴/상의/하의/신발(+α)
- **동작 시스템** 🎬: 걷기 / 달리기 / 춤 / 인사 / 점프 + **🔁 연속 재생**(4초마다 자동 전환)
  - 착용 상태 그대로 팔다리 관절 스윙 — 소매·바지·신발·손 아이템이 팔다리와 함께 움직임
  - 하의는 clipPath로 허벅지(y=270)에서 좌우 분할, 치마류는 `rigid: true`로 통짜 고정
- **코인 경제**: 시작 5,000코인 → 구매 → 자동 장착, 장착/해제 토글
- **인터렉티브** 🎮
  - 눈동자가 마우스 커서를 따라옴
  - 2~6초마다 자연스러운 눈 깜빡임
  - 캐릭터 클릭 → 점프 + 코인 +10 (10% 확률 행운 +100)
  - 장착 시 팝 애니메이션, idle 둥실거림
- **자동 저장**: localStorage (`avatar-shop-v1`) — 새로고침해도 유지

## 구조

| 파일 | 역할 |
|---|---|
| `index.html` | 레이아웃 (무대 + 얼굴/상점/인벤토리 탭) |
| `css/style.css` | 스타일 + 애니메이션 (idle/jump/pop/blink) |
| `js/data/face.js` | 눈·코·입 카탈로그 (24종) |
| `js/data/items.js` | 아이템 카탈로그 (120종, 슬롯/가격/SVG) |
| `js/state.js` | 코인·보유·장착 상태 + localStorage |
| `js/character.js` | SVG 레이어 합성 렌더러 |
| `js/shop.js` | 상점/인벤토리/얼굴 탭 UI |
| `js/interactive.js` | 눈동자 추적·깜빡임·클릭 보상 |
| `js/main.js` | 초기화 |

## 좌표계 (아이템 추가 시)

viewBox `0 0 320 400` 기준 — 머리 중심 `(160,120)` r56, 눈 `(140/180,110)`,
손 `좌(103,252)` `우(217,252)`, 바닥 `y=360`. 새 아이템은 `js/data/items.js`에
`{ id, name, price, svg }` 형태로 추가하면 상점에 자동 노출됩니다.
